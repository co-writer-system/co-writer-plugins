# Co-Writer System Plugin Marketplace

A living library of AI co-writing tools for Claude Cowork. Built for content creators, writers, and professionals who use Claude as their production system.

Part of [The Co-Writer System](https://alexmcfarland.ai/co-writer-system) by Alex McFarland.

## Installation

1. Open **Claude Cowork** (or Claude Desktop)
2. Go to **Plugins** ‚Üí **+** ‚Üí **Add Marketplace from GitHub**
3. Paste: `co-writer-system/co-writer-plugins`
4. Click **Sync**

All plugins will be available as slash commands in your Claude sessions.

## Plugins

### Setup

Get your workspace configured and keep it current.

- `/setup` ‚Äî Build your CLAUDE.md and workspace structure
- `/update` ‚Äî Update your configuration to the latest version

### Voice

Capture and maintain your authentic writing voice.

- `/voice-create` ‚Äî Build a voice profile from writing samples
- `/voice-audit` ‚Äî Evaluate how well content matches a voice profile
- `/voice-update` ‚Äî Refine a voice profile with new samples

### YouTube

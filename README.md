# Co-Writer System — Plugin Marketplace

A growing library of AI co-writing plugins for Claude. Built for content creators, writers, and professionals who use Claude as their production system — no coding required.

Part of [The Co-Writer System](https://alexmcfarland.ai/co-writer-system) by Alex McFarland.

## Installation

1. Open **Claude Cowork** (or Claude Desktop)
2. Go to **Plugins** → **+** → **Add Marketplace from GitHub**
3. Paste: `co-writer-system/co-writer-plugins`
4. Click **Sync**

All plugins install at once and stay updated automatically. Every skill becomes a slash command in your Claude sessions.

## Connectors

Some plugins work best with MCP connectors for live data. These are optional — every skill works without them, but capabilities expand when connected.

| Connector | Used by | What it adds |
|-----------|---------|-------------|
| Perplexity | Research, AEO | Live web search with citations, `related_questions` chaining |
| Firecrawl | Research, AEO, Blog | Web scraping, site crawling, structured data extraction |
| YouTube | YouTube | Channel stats, video search, comments, playlists |
| Substack | Substack | Post analytics, email stats, subscriber growth |

Connectors are available through the Co-Writer System platform at [app.alexmcfarland.ai/connectors](https://app.alexmcfarland.ai/connectors).

# Landing Page Plugin

Landing page copy and production-ready page builds. Write the copy, then build the actual HTML/CSS page — or do both at once.

## Skills

| Skill | What it does |
|-------|-------------|
| sales-page | Writes long-form sales page copy with hooks, benefits, objections, and CTAs |
| product-page | Writes product page copy focused on features, use cases, and positioning |
| opt-in-page | Writes lead capture page copy for email signups and free offers |
| build-page | Builds a production-ready HTML/CSS landing page — conversion structure + high-end design in a single deployable file |

## Workflow

The copy skills (sales-page, product-page, opt-in-page) produce markdown. The build skill (build-page) produces actual HTML/CSS pages. You can use them together — write copy first, then build — or go straight to build-page if you want both copy and design in one step.

No connector dependency. No frameworks required. Pages output as single HTML files you can open in any browser.

# Setup Plugin

System setup and maintenance for the Co-Writer System.

## Skills

| Skill | What it does |
|-------|-------------|
| co-writer-setup | Walks through full system setup — CLAUDE.md, workspace folders, and connector configuration from scratch |
| co-writer-audit | Audits your current setup and flags missing, thin, or outdated sections |
| co-writer-update | Updates and strengthens your CLAUDE.md — runs an internal audit then walks you through fixing every weak section |

# Kit

Kit (ConvertKit) email sequences — sales campaigns, launch sequences, and nurture series.

## Skills

| Skill | What it does |
|-------|-------------|
| sales-sequence | Multi-email sales campaigns with urgency and objection handling |
| launch-sequence | Product or feature launch email series |
| nurture-sequence | Relationship-building sequences for new subscribers |

## Connector

This plugin works standalone, but unlocks more when connected to **Kit**.

| Feature | Without Connector | With Connector |
|---------|------------------|----------------|
| Sequence writing | Based on CLAUDE.md context | Informed by existing broadcast stats and open rates |
| Subject lines | Best-practice formats | Modeled on your highest-performing sends |
| Segmentation | Generic audience targeting | Uses your actual tags and subscriber segments |

**Setup:** Available at app.alexmcfarland.ai/connectors. Requires your Kit API key (starts with ck_) from app.kit.com/settings/developer.

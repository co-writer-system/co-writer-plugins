# Blog Plugin

Long-form articles and SEO content.

## Skills

| Skill | What it does |
|-------|-------------|
| long-form-article | Writes in-depth articles from a topic or outline |
| how-to-article | Writes step-by-step tutorial articles optimized for search |
| seo-optimizer | Audits and improves existing content for search performance |

## Connector

This plugin works standalone, but unlocks more when connected to **Perplexity**.

| Feature | Without Connector | With Connector |
|---------|------------------|----------------|
| Research depth | Writes from your expertise and provided context | Researches current sources, stats, and competitor content before writing |
| Citations | No external citations included | Pulls live web sources and includes proper citations throughout |
| SEO optimization | Optimizes based on keyword best practices | Analyzes what's currently ranking and structures content to compete |

**Setup:** Connect at app.alexmcfarland.ai/connectors

# YouTube Plugin

Scripts, titles, and descriptions for YouTube videos.

## Skills

| Skill | What it does |
|-------|-------------|
| youtube-script | Writes a full video script from a brief or topic |
| youtube-titles | Generates title options optimized for clicks and clarity |
| youtube-description | Writes a YouTube description with timestamps, links, and keywords |

## Connector

This plugin works standalone, but unlocks more when connected to **YouTube**.

| Feature | Without Connector | With Connector |
|---------|------------------|----------------|
| Script writing | Writes based on your CLAUDE.md context and topic brief | Pulls your actual video stats and top comments to inform tone, length, and what resonates |
| Title generation | Generates titles from topic and positioning | Analyzes your best-performing titles and models new ones on what already works |
| Description writing | Writes descriptions from script content | References your channel stats and existing video structure for consistency |

**Setup:** Connect at app.alexmcfarland.ai/connectors

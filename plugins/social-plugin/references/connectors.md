# Available Connectors for Social

Social content skills can benefit from the Substack connector for Notes performance data. Check if Substack MCP tools are available before using them — if they're not, the skills still work without them.

## Substack (for Substack Notes)

| Tool | What it does |
|------|-------------|
| list_published_posts | See recent posts including Notes |
| get_post_analytics | Check engagement on Notes |

## How Skills Should Use These

- **Before writing Substack Notes:** Check which recent Notes got the most engagement. Use patterns from top performers.

## If No Connectors Are Available

The skills work without any connectors. Write based on the context in their CLAUDE.md and any information they provide.

## Setup

The Substack connector is available through the Co-Writer System platform at app.alexmcfarland.ai/connectors.

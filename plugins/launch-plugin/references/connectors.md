# Connector Reference: Launch Plugin

## Recommended: Kit Connector

This plugin works standalone — all skills produce email copy in markdown regardless of what email platform you use. But connecting Kit unlocks more.

| Feature | Without Connector | With Kit Connector |
|---------|------------------|-------------------|
| Sequence writing | Based on CLAUDE.md context | Informed by existing broadcast stats and open rates |
| Subject lines | Best-practice formats | Modeled on your highest-performing sends |
| Segmentation | Generic audience targeting | Uses your actual tags and subscriber segments |
| Launch ops | Manual copy-paste to email platform | Can reference existing sequences and broadcasts |

**Setup:** Available at app.alexmcfarland.ai/connectors. Requires your Kit API key (starts with ck_) from app.kit.com/settings/developer.

**Not using Kit?** No problem. The email sequences this plugin produces are platform-agnostic markdown. Copy them into whatever email tool you use — Mailchimp, Beehiiv, Loops, ActiveCampaign, or anything else.
